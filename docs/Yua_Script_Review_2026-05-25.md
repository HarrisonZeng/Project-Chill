# Yua 当前剧本审阅与扩写建议（2026-05-25）

## 审阅范围

- 当前应当作为主要修改对象的是 `data/dialogue/scripted_nodes.json`，版本为 `demo_script_version: 7`，包含 `Ep 0` 与 `Ep 1` 到 `Ep 14`。
- `docs/Yua_Script_Draft_ZH.md` 是更早、更偏主线展开的草稿，可作为点子库，但不应直接覆盖当前慢热版本。
- 已通读参考脚本 `tmp/cwyl_extract/clean/scenario_readable.txt`：首次连接、主线章节、番外、小聊、专注开始/结束、点击、回归、离开与节日台词均纳入比较。
- 本文只提出台词与结构建议，不直接改写运行中的 JSON。

## 总体判断

Claude 这版改动的方向是对的：把“一上来就暴露创作主线”收住了，变成先认识、一起坐着、慢慢熟起来；也把任务改成可选，不再像被 Yua 带着完成流程。相比早期草稿，它更符合“陪伴优先”的产品方向。

现在最需要提升的不是增加大事件，而是让每次专注后的短谈真正像一个小小奖励：

1. **每个 episode 目前太快收束。** 大多数只有 Yua 起一个话题、玩家点一次、Yua回答一次就结束，玩家刚结束一段专注时，会觉得奖励刚开始便断掉了。
2. **少数口语不够自然，或显得过于刻意可爱。** 如“坐进去”“压得住”“待不腻”“配上就固定了”等，更像撰写概念台词而不是一个女孩当下会说的话。
3. **对话入口经常立刻问玩家刚才做得如何。** 玩家刚结束专注，可能累、松了一口气、甚至觉得没做够；让 Yua 先分享她自己的小瞬间，再把回应权交给玩家，会更放松。
4. **结尾大多只是空 choices 结束。** 逻辑上回到共处没问题，但情绪上缺一个“收好这几句话，再安静待在一起”的落点。
5. **`ep13` 有明确连续性风险。** 她在没有真实记忆数据保证的情况下说会记住玩家提过的小事，违反“不可伪造记忆”的规则。
6. **主线克制是优点，应继续保持。** 当前 live JSON 到 `ep05` 才只说“会写点东西”，比旧稿过早公开月面便利店更适合“主线 TBD、普通聊天优先”的现阶段。

## 从 CWYL 学什么，又不要照搬什么

### 值得学的基础功

- 它让普通聊天也有内容：背景声、吃的东西、词语、天气、杯子、散步、肩膀酸等都能形成一小段完整谈话。
- 它的章节常有 8 到 18 个短拍子：起因、走神、具体细节、小自嘲、回到工作。话题不是只有“主题句 + 结论句”。
- 它有很多情境台词池：开始、短时完成、长时完成、中断、不同时间、点击与离开，所以长时间使用不容易只看到同一句话。
- 它懂得让角色先讲自己的生活，再轻轻邀请玩家回应；玩家不是每次都要汇报生产力。

### Project Chill 应当做得更好的地方

- CWYL 有不少“你有没有完成 / 你得继续 / 不许偷懒”式话语。Yua 应更像同桌伙伴：她可以被玩家抓到自己也走神，然后一起回到安静，而不是检查玩家。
- 不需要沿用参考作品的科幻系统谜团或强情感危机。Yua 的独特性先来自她的口吻、她的小习惯、她看世界的方式。
- Project Chill 可以用可选任务、真实记忆、Type Mode 和更细腻的回归语气，做到“她记得我，但没有盯着我”。
- 可爱不应主要靠感叹号或撒娇，而应来自具体而轻的反应，例如捂住刚说漏嘴的文件名、承认把杯子摆到镜头里只是因为好看、一本正经为一块小饼干辩护。

## Yua 口吻准则

### 应保留的气质

- 安静、带一点停顿；温暖但不灌鸡汤。
- 偶尔自嘲，尤其是关于她自己分心、拖延、讲太多小事。
- 会轻轻逗玩家，但笑点落在“我们都有点这样”，不落在“我管你”。
- 在熟悉之后可以更活泼一点，但不要突然变成黏人恋爱路线。

### 让中文更自然的调整表

| 当前倾向 | 问题 | 更自然方向 |
| --- | --- | --- |
| “你一坐进去” | “坐进去”不像自然口语 | “你一开始，我这边也跟着安静下来了” |
| “我反而压得住” | 略生硬，像说明机制 | “我反而没那么容易飘走” |
| “我怎么都待不腻” | 亲密度推进太猛 | “这种节奏……我好像越来越喜欢了” |
| “反正这软件配上就固定了” | 像系统说明，破坏情绪 | “下次亮起绿点的时候，我应该就知道是你了” |
| “你人在这儿，就比躲着不做强” | 会让失败感更重 | “这一段不用打分。你回来了，就好” |
| “非常有文学气质的进展” | 旧稿有趣，但略写作腔 | “总之，光标真的往前挪了两行。请给它一点面子” |
| “有点上瘾了” | 第一次完成便过度靠近 | “这种感觉……比我预想的舒服” |
| “一直都是你了” | 过早带永久/占有意味 | “看来以后还会常常见到你” |

### 可爱的强度控制

- 初识：可爱来自紧张后的自我化解，例如“我刚刚是不是对着自己的小窗口说了半天话……算了，当作热身。”
- 熟悉：可爱来自共享的小秘密，例如“我把今天的进度写成两行，显得它很像一件大事。”
- 更熟之后：可爱来自轻微理直气壮，例如“这杯水现在是我的专注道具，不接受质疑。”
- 避免连续使用“嘿嘿、哈哈、呀、啦、诶”来制造可爱；每段最多一处明显轻快句尾即可。

## 一段奖励对话的推荐节奏

每次新 episode 最适合在完成专注后的自然休息或玩家点击 Yua 时出现，而不是强制打断。

### 自然入口

先承接玩家刚结束一段专注的身体与心理状态，再让话题露出来：

1. 计时结束的共同反应：“……响了。等一下，我刚好把这一句收住。”
2. 给玩家缓气空间：“你先松口气，不用马上回答我。”
3. Yua 自然冒出话题：“我刚才发现一件有点好笑的小事。”

这样比开场就问“你刚才专注得如何？”更像休息中的聊天。

### 合适长度

- 普通奖励 episode：`开场 2-3 句 -> 玩家选择 -> Yua 3-4 句 -> 可选追问/回应 -> 轻落点 1-2 句`。
- 总量约 7-11 句 Yua 台词，阅读约 35-60 秒。
- 重要但仍克制的节点（第一次透露写作、习惯形成）：约 10-15 句，可有两个玩家选择点。
- 不必每次都聊满；重复完成用短台词池，新的 episode 才提供完整小聊。

### 自然结尾

每段结束应让玩家感到“聊天结束了，但陪伴还在”，而不是故事窗口忽然没了：

- “好，不继续占用你的休息时间了。我把这边也保存一下。”
- “你先缓一会儿。我还在这边，想再开一段的时候再叫我。”
- “话说完了，杯子也快凉了。我们先安静一会儿。”
- “我把这个念头记下来。今天它能出现，已经挺好了。”

避免每次都直接推下一段专注；计时按钮本来就在 UI 中，Yua 不需要主持流程。

## 逐 episode 审阅：Ep 1 至 Ep 14

### `ep01_01`、`ep01_a/b/c`：第一段

**保留：** 第一段完成后询问感受是合理的，三种回答覆盖了舒服、普通与分神。

**问题：** 第一件奖励仍然由玩家汇报状态开场，且 `a` 的“上瘾了”推进太快，`c` 的“躲着不做”带评价。

**扩写方式：** 让 Yua 先承认自己也刚放下紧绷的肩膀，并说她本来担心视讯里会很尴尬；玩家回答后，她再发现“各忙各的也能舒服”。结尾落到第一次形成的安静感。

**示例收尾：**

> Yua：我本来还怕第一次通话会一直想找话说。  
> Yua：结果安静下来以后，好像也不需要证明什么。  
> Yua：……挺好的。你先休息，我也把刚才那一点存下来。

### `ep02_01`、`ep02_a/b/c`：放点什么

**保留：** 非剧情的生活话题非常适合第二次奖励，也贴合音乐 UI。

**问题：** 与 CWYL 的早期背景音话题过于近似；`a` 结尾又回到“你让我能做事”的功能总结。

**扩写方式：** 变成 Yua 自己的“奇怪背景音习惯”，例如她会把雨声开太大，后来分不清窗外是否真的下雨；或她收藏了一个只有翻页声的音轨。让玩家选“音乐 / 环境声 / 安静”，最后由 Yua 提议某天交换一段“不妨碍工作的声音”。

### `ep03_01`、`ep03_a/b`：坐不住

**保留：** 开始碰触她为什么来这里，但仍是普通共感。

**问题：** “俩坐不住的人互相拽着，谁也别想偷偷溜号”像监督协议；`b` 的“蹭一蹭你定力”也把玩家过度理想化。

**扩写方式：** 写她自己的小动作：鼠标绕去别的标签页又悄悄回来、拿起手机又反扣回桌面。玩家不是拽住她，而是让她意识到“回到文档也没那么困难”。

### `ep04_01`、`ep04_a/b`：换换脑子

**保留：** 这是很好的普通聊天题材，并且已有 Type Mode 开口让玩家分享习惯。

**问题：** “怎么熬过去”“硬扛派……强”稍硬，像对策略评语；分支结束太急。

**扩写方式：** 将问题换成“卡住的时候，你会偷偷做什么？”Yua 可承认她会把杯垫挪正、给文件换一个太认真的标题。玩家回复后，加一句她笑自己已经休息够了，但不急着要求玩家继续。

### `ep05_01`、`ep05_a/b`：我也写点东西

**保留：** 现在只揭露“写东西”，没有锁定主线内容，节制非常好。

**问题：** 作为第五次完成的奖励，信息量略薄；她一句“还不能说”以后话题就关闭了。

**扩写方式：** 不揭露作品设定，但给具体可爱的边缘信息：她的文件名越改越心虚，或她今天终于把一个空文件打开了。玩家可以问“连类型也不能说吗？”或说“不急，我等你想说时再说”。她可以回一句半遮半露的细节，如“至少里面没有工作报表……大概。”

**连续性建议：** 这里应设置 `writing_disclosed` 或等价 flag，供 AI prompt 使用，避免 Type Mode 在此之前擅自聊她的写作，也避免之后仍假装没说过。

### `ep06_01`、`ep06_a/b`：写不动

**保留：** “越重视越下不去手”是真实而有共鸣的情绪。

**问题：** 它紧跟 Ep 5，又继续谈写作低潮，普通聊天的比例立刻被挤小；“卡死了”“光想逃”比整体气氛重。

**建议：** 若坚持主线慢热，将本话题移动到更后面；Ep 6 改成轻生活聊题。若保留，则减轻为她在一个句子前来回移动光标，并在结尾发现今天只改标点也不算坏。

### `ep07_01`、`ep07_a`：喜欢的词

**保留：** 很适合一个写东西但尚未讲主线的 Yua，也适合 Type Mode。

**问题：** “余白”“松弛”并排略像品牌文案，不像她突然想分享的小怪癖。

**扩写方式：** 让她喜欢词的声音或形状，例如“慢慢来”“刚刚好”“咔哒”；也可以说她今天卡在一个词上，读了三遍以后忽然开始觉得它很好笑。若玩家输入一个词，AI 只需轻轻回应为何听着舒服，不要分析人格。

### `ep08_01`、`ep08_a/b`：灵感

**保留：** 把玩家的存在与她的小进展联系起来，符合长期回报。

**问题：** “算你一半功劳”“第一个谢的就是你”已经像大型故事回报，早于主线确认；灵感也过于抽象。

**扩写方式：** 她不必说出写了什么，只描述一种小成功：“有一句之前总是站歪，刚才突然肯好好待着了。”把感谢写成玩笑式的小记号：“我在旁边画了个很小的星号，算今天这个通话的。”

### `ep09_01`、`ep09_a/b`：拖延

**保留：** 能让玩家觉得自己并非唯一会有空转日的人。

**问题：** “一整天啥正事没干”“心里坠着块石头”“天天如此”连续堆叠，有向负面倾斜的风险；`b` 把不拖延玩家架成榜样。

**扩写方式：** 改成 Yua 昨天把计划写得很漂亮，却只做了最无关紧要的一格；今天她笑着把计划缩成一行。这样仍承认逃避，但不沉重。对不拖延玩家，可回应“那你很会照顾自己的节奏，我先学着把今天这一小格做好。”

### `ep10_01`、`ep10_a/b`：习惯你在

**保留：** 这是前十次共处后的重要感情回报，主题正确。

**问题：** “自在得跟呼吸似的”“这个位置，给你留着”亲密强度偏高，也偏 VN 告白式。

**扩写方式：** 用具体变化证明熟悉：她不再上线前反复检查头发、不会因安静而拼命找话、看到绿点就顺手把水杯摆好。玩家会更自然地感觉关系变近。

### `ep11_01`、`ep11_a`：划掉那一下

**保留：** 她主动来分享小开心，很贴合“奖励”。

**问题：** 当前只有概念，没有那件“小事”的纹理，所以容易与通用励志句混在一起。

**扩写方式：** 让她说出不涉及主线的小项目：整理了一页乱掉的笔记、回复了一封拖着的邮件、把乱取的文件名改回能看的名字。玩家可写自己的小完成，也可夸她；她的结尾可以是把勾号画得过大，自己有点得意。

### `ep12_01`、`ep12_a/b`：敢读回去

**保留：** 目前 14 段里回报感最强的一个：它具体、脆弱但不沉重，而且能看见 Yua 前进。

**问题：** 仍然只有一轮回应；“勇敢”分支稍郑重。

**扩写方式：** 加一个可爱的自保动作：她本来把删除键按在手边，最后只删了两个多余的逗号。玩家若夸她，她先说“请不要突然给我颁奖”，再诚实接受那点开心。

### `ep13_01`、`ep13_a`：记得你

**保留：** 真正用到玩家记忆时，这会是 Project Chill 超过参考作品的核心亮点。

**严重问题：** 当前节点按专注次数解锁，不保证有真实记忆，却直接暗示她记得玩家说过的小事；这违反 memory fail-safe，也会让玩家觉得被假装熟悉。

**必须调整：**

- 如果没有 validated memory：把本 episode 改成一般的“我开始记得我们的节奏”，不提玩家曾说过具体事情。
- 如果有真实 memory：由系统注入一个很轻的具体回声，例如“你上次提过的那门课，今天也在忙它吗？”只说一个事实，不总结玩家生活。
- 无论哪种路径，玩家都可以不回答；Yua 不追问。

### `ep14_01`、`ep14_a`：固定搭子

**保留：** 作为一轮普通章节的收束，承认关系已经熟悉是合理的。

**问题：** “往后大概，一直都是你了”“我珍惜”叠在一起偏恋爱化，也把 app 配对机制说得太直；作为长期 chill 游戏，这种承诺发得太早会限制之后升温空间。

**扩写方式：** 将回报放在日常习惯上：“现在我上线看到你，就知道不用急着想开场白了。”可让 Yua 给这段关系起一个轻轻的称呼，如“固定安静席位”，而不是永久承诺。

## 优先应重写的五个位置

### 1. 首次结尾 `ep00_close`

目标：玩家不被任务推着走，却愿意留下。

```text
Yua：好，那我记住了。

Yua：我先把耳机戴好，继续忙我这边的。

Yua：你想开一段计时就开；不想的话，也可以先安静坐一会儿。

Yua：……第一次这样跟人待着，好像也没那么奇怪。
```

### 2. 第一段完成 `ep01`

目标：先照顾完成后的松气感，再谈共处。

```text
Yua：……响了。

Yua：我刚好把光标停在一句还算能看的地方。今天运气不错。

Yua：你先缓口气。刚才那一段，对你来说还好吗？
```

在玩家回答后：

```text
Yua：嗯。我本来担心通话里一直安静会很尴尬。

Yua：结果你在那边忙，我在这边忙，反而比硬找话说舒服。

Yua：先把这份舒服留一会儿吧。我不催你开始下一段。
```

### 3. 写作轻透露 `ep05`

目标：给奖励感，但不锁死主线题材。

```text
Yua：刚才……我也偷偷做了一件拖了有点久的事。

Yua：我把一个写东西用的文件打开了。

Yua：它的文件名现在有点丢脸，所以暂时不能给你看。
```

玩家问内容后：

```text
Yua：内容也先保密。

Yua：不是故意吊你胃口，是我一说出口，就会突然觉得它很需要被删掉。

Yua：等它胆子大一点……或者等我胆子大一点，再告诉你。
```

### 4. 记忆节点 `ep13`

目标：只有真实记忆才出现具体回声。

无真实记忆版本：

```text
Yua：最近我发现，我们已经有一点固定节奏了。

Yua：不是发生了什么大事。就是你出现、我们各自安静一阵、偶尔说两句话。

Yua：这种小小的熟悉感……我还挺喜欢的。
```

有真实记忆版本由数据插入一句具体且克制的问候，再回到安静共处。

### 5. 第十四段结尾 `ep14`

目标：熟悉、可爱、留有成长空间。

```text
Yua：我们这样坐过好多次了，对吧。

Yua：刚开始我还会偷偷检查自己是不是显得很尴尬。

Yua：现在看到你这边亮起来，我就会很自然地把文档和水杯都放好。

Yua：看来你已经变成我的固定安静席位了。……这个称呼是不是有点怪？
```

玩家接受后：

```text
Yua：那就先不改名了。

Yua：你忙你的，我忙我的。偶尔抬头的时候，知道对方还在就好。
```

## 新增普通聊天 episode 点子

这些 episode 不推进主线，只给玩家“我又多认识 Yua 一点”的轻奖励。可以插在写作相关段落之间，避免连续多集都围绕她卡稿。

### 新 Ep A：杯子留下的圆印

**适合位置：** Ep 3 与 Ep 4 之间。  
**主题：** 桌面的小习惯，轻而生活化。  
**开场：**

```text
Yua：刚才抬杯子的时候，我才发现桌上留下一个特别完整的圆印。

Yua：完整到我有点不舍得擦掉。像今天这一段的盖章。
```

**玩家选项：** “那就留着。” / “擦掉吧，会越来越多的。”  
**结束：** 她决定拍一下再擦，并说“好了，证据保存，桌子也获救了。”

### 新 Ep B：不合时宜的小零食

**适合位置：** Ep 6 附近，替代一集沉重的卡稿。  
**主题：** 休息后的嘴馋，带可爱但不恋爱。  
**开场：**

```text
Yua：我有个很严肃的问题。

Yua：专注结束以后突然想吃脆的东西，是不是很合理？
```

**玩家选项：** “非常合理。” / “你只是饿了。” / “你吃什么？”  
**结束：** 她发现饼干包装会吵到麦克风，郑重承诺“我会静音作案。”

### 新 Ep C：文件名博物馆

**适合位置：** Ep 5 写作透露之后，但不揭露内容。  
**主题：** 她乱取文件名的习惯。  
**开场：**

```text
Yua：我刚才整理文件，发现我以前真的很不会取名字。

Yua：有一个叫“这个大概真的能用_改二_不要再改了”。
```

**玩家选项：** “听起来还会再改。” / “至少找得到。”  
**结束：** “我给它改了个正常名字。成熟的大人今天完成了一件事。”

### 新 Ep D：窗口里的天气

**适合位置：** 任意傍晚/夜间完成后的随机 episode。  
**主题：** 通话空间感与现实时间。  
**开场：**

```text
Yua：我刚刚才发现，你来的时候这边还亮着。

Yua：一段计时过去，窗上已经只剩台灯的倒影了。
```

**玩家选项：** “时间过得好快。” / “夜里更适合安静做事。”  
**结束：** 她不浪漫化熬夜，只说“这种光挺安静的。再待一会儿也好，想收工也好。”

### 新 Ep E：输入法背叛事件

**适合位置：** 中前期通用奖励。  
**主题：** 工作时发生的小尴尬，体现她活着而非只谈感受。  
**开场：**

```text
Yua：我刚才想记一句很正经的话。

Yua：结果输入法把它改成了一句完全不像人类会写的东西。

Yua：我盯着它三秒，差点觉得错误版本更有灵气。
```

**玩家选项：** “保留错误版本。” / “请尊重原句。”  
**结束：** 她偷偷把错句存进“也许有用”的角落，再继续正经工作。

### 新 Ep F：耳机压乱的头发

**适合位置：** 熟悉度稍高、但仍是日常闲聊时。  
**主题：** 视频通话独有的小真实感。  
**开场：**

```text
Yua：等一下，我是不是被耳机压出了一块很奇怪的头发？

Yua：……算了，你就当这是专注留下的痕迹。
```

**玩家选项：** “看不出来。” / “有一点可爱。” / “非常明显。”  
**结束：** 无论分支都让她笑着理一下，然后说“好了，处理完毕。刚才那段也算值得了。”

## 建议的内容编排

现在 `ep01` 到 `ep14` 每完成一次就触发一集。若玩家选择很短的 session，Yua 的关系会升温得太快。建议将内容分成三层：

1. **每次完成都可能出现的短休息反应池：** 水杯、伸展、保存文件、窗外光、她刚好停笔等，一到三句。
2. **普通聊天 episode：** 以上新增点子与背景音、卡住小习惯、喜欢的词等，约每若干次完成或按未读池出现。
3. **关系/写作回报 episode：** `ep05`、`ep10`、`ep12` 等，只按累计专注时间与完成次数慢慢解锁。

这样玩家会持续有新鲜回应，但不会因为连续做几段短计时就迅速走完 Yua 的熟悉度。

## 内容与系统需要对齐的项目

- `ep00_name` 应提供跳过昵称路径，或者明确昵称是可空的；首次体验不应被必填信息拦住。
- `ep05` 首次承认写东西后，应设置可读的 `writing_disclosed` flag；AI 世界设定必须受它约束。
- `ep13` 只有在存在真实 validated memory 时才能使用具体记忆表述；否则改为一般熟悉感 episode。
- 完成后的通用台词应按短段、普通段、长段、深夜与连续专注分池，避免所有 session 都出现同一个自我评价问题。
- 每个 episode 末端应有“回到安静共处”的一条自然落点，配合引擎把 terminal node 正确归回 `idle`。

## 推荐下一步

先不急着写主线设定。优先完成一次“口吻与奖励感”改写批次：

1. 改写 `ep00`、`ep01`、`ep05`、`ep13`、`ep14`，这五处最影响第一印象、首次奖励、连续性与关系升温。
2. 为 `ep02` 到 `ep12` 每集增加一个中间回应和一个自然收尾；同时将 `ep06` 换成更轻的普通聊天，或推迟写作低潮。
3. 增加至少 12 条完成/休息短池与 4 个普通聊天 episode，再决定 Yua 作品内容是否正式定稿。

这会先把“完成一段专注，想点她聊一下”的基础体验做好，再留足空间发展真正独有的故事。
