# Project Chill Dialogue Viewer

Open `dialogue_viewer.html` in a browser and load `data/dialogue/scripted_nodes.json`.

Main uses:

- visualize dialogue node links
- inspect choices, unlock gates, flags, speaker, and tags
- find missing links and orphan nodes
- let non-programmers suggest edits/comments in review mode
- export review notes, revised JSON, Mermaid flowcharts, or a shareable HTML snapshot

See `docs/Dialogue_Review_Guide.md` for friend-facing instructions.
